# Introduction

## Installation

You can add the Zyle CSS library in these ways:

### Install from CDN

```html
<link rel="stylesheet" href="https://unpkg.com/zyle.css/css/zyle.min.css" />
```

### Install from NPM

```bash
npm install zyle.css
```

### Install from Yarn

```bash
yarn add zyle.css
```
